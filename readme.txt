== Description ==
Post Type Slider Widget provides widget for post and custom post type featured images slider. Post Type Title and Featured Image is display in your sidebar in slider form. It Supports your custom post type.

== Installation ==
Upload the post-type-slider-widget folder to the /wp-content/plugins/ directory. OR add new plugin from WordPress admin. 
Activate the plugin through the ‘Plugins’ menu in WordPress.
